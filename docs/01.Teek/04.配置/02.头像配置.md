## logo 配置

网站的 Logo 图标还没有，参考下方目录表

```md{6}
.
├─ Demo
│ ├─ docs-base
│ │ └─ .vitepress
│ └─ public <--静态资源目录
│ │ └─ favicon.ico <--logo
│ └─ index.md
└─ package.json
```

按照目录新建 public 文件夹(如有跳过),在 Demo\docs-base\public 中 放入网站 logo，命名为 favicon.ico

然后在 `Demo\docs-base\.vitepress\config.mts` 中配置

```ts{3}
export default defineConfig({
  // 网站logo  [!code focus:2]
    ["link", { rel: "icon", href: "/public/favicon.ico", type: "image/png" }],
});
```

## 头像配置

- 在 `Demo\docs-base\.vitepress\config.mts` 中配置

```ts{4}
export default defineConfig({
  themeConfig: {
    // https://vitepress.dev/reference/default-theme-config     [!code focus:1]
    logo: "https://hyde.seasir.top/img/hyde-logo.ico",  //左上角头像
  },
});
```
